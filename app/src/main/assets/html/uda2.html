Поплаво́чная у́дочка — одна из простейших рыболовных снастей, что обусловило её повсеместное распространение. Поплавочная удочка состоит из удилища, закреплённой на нём лески с поплавком, грузилом и крючком с приманкой. С помощью удилища приманка забрасывается в воду. По поведению поплавка (поклёвке) определяется момент, когда рыба держит приманку во рту. При поклёвке производится подсечка — небольшое резкое движение удилищем. Подсечённая рыба вываживается на берег.
